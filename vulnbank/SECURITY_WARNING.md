# SECURITY WARNING

## This Project Is Intentionally Vulnerable

`VulnBank` is a deliberately insecure demo application built for:

- Security education
- Static analysis scanner testing
- Vulnerability pattern demonstrations

## Do Not Deploy

- Do **NOT** deploy this project to production.
- Do **NOT** expose this project to the public internet.
- Do **NOT** reuse this code in real systems.

## Scope and Purpose

All weaknesses are intentionally included to help tools like **Needle** detect issues such as:

- Injection risks
- Hardcoded secrets
- Broken access control
- Weak authentication practices
- Insecure runtime configuration
- XSS-prone rendering patterns

Use only in isolated local/dev environments.
